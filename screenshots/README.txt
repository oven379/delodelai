Скриншоты для лендинга «Дело»

Положите сюда 8 скриншотов приложения с именами:

  1.png — Главный экран «Сегодня», тёмная тема (список дел, прогресс 3/4)
  2.png — Меню «Настройки» (тема, режим списка, уведомления), тёмная тема
  3.png — Меню «Настройки», светлая тема (опционально)
  4.png — Главный экран «Сегодня», светлая тема
  5.png — Экран «Завтра», светлая тема (опционально)
  6.png — Панель «Уведомления» (сработавшие напоминания), светлая тема
  7.png — Панель «Уведомления», тёмная тема (опционально)
  8.png — Экран «Завтра», тёмная тема

Формат: PNG или JPG. Соотношение сторон лучше 9:19 (как экран телефона).

Чтобы подогнать скрины ровно под рамки телефонов на лендинге, запустите обрезку до соотношения 9:19.5:
  node crop_screenshots.js   (нужен Node.js и sharp: npm install sharp)

На лендинге используются в основном: 1, 2, 4, 6, 8. Остальные можно добавить при желании.
