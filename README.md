# Лендинг «Дело»

Статичный лендинг: HTML, CSS, скриншоты. Сборка не требуется.

**Как открыть:** откройте в браузере файл `index.html` из этой папки или поднимите локальный сервер (например `npx serve .` в папке landing).

**Скриншоты:** папка `screenshots/`. Список имён и назначения — в `screenshots/КАКИЕ_СКРИНЫ_НУЖНЫ.txt`.

**Обновление кэша:** при изменении стилей увеличьте параметр в `index.html`: `styles.css?v=7` → `?v=8`.
